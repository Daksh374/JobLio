# 💼 JobLio – Job Portal Web Application

JobLio is a full-stack job portal web application that connects job seekers with recruiters through a simple and efficient platform. Users can explore job listings, apply for positions, and manage their profiles, while recruiters can post and manage job opportunities.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- Browse and search for jobs
- Apply to job listings
- View applied jobs
- Profile management

### 🏢 Recruiter Features
- Post new job listings
- Manage job postings
- View applicants

### 🔍 Job Search
- Search jobs by title, location, or keywords
- Filter jobs based on requirements
- Dynamic job listing display

### 📊 Dashboard
- User dashboard for tracking applications
- Recruiter dashboard for managing jobs
